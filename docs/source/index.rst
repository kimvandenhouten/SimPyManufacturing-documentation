Welcome to the SimPyManufacturing documentation!
===================================

**SimPyManufacturing** is a tool that uses the SimPy library for Discrete-Event-Simulation to model highly-complex, multi-product, multi-stage manufacturing systems that are representative for process industries, such as biomanufacturing. 

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
